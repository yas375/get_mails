get_mails by yas375

Парсер email'ов владельцев доменов.

На входюе сркипту задаём список доменов. Для доменов, с которыми скрипт знаком он получает whois информацию через консольную утилитку whois. И затем по определённым правилам для этого домена "вытягивает" оттуда информацию об email.
На выходе выдаётся статистика о проделанной работе и файл в формате csv. Файл этот можно открыть в excell и там будет табличка, шдже в одной колонке домен, а в другой email адрес.

Dependencies

linux
php + apache
whois

Install

На папку data должны быть права на запись. Сделать это можно командой:
chmod 777 data/
